dcmGridRowActions

  • - directive in module dcm-ui.grid

The contents of this element is transcluded into the row under the active grid row (spanning all the cols). The parent dcm-grid thus must have the active-row attribute specified. Also the data from the row is transcluded into the scope of this row. Because this is transcluded/compiled it may contain additional angular elements.

Directive Info

  • This directive executes at priority level 0.

Usage

<dcm-grid-row-actions>{{really_long_description_field_in_row_data}}</dcm-grid-row-actions>
